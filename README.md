# Node.jsの本

この本では、nodejsのAPIの使い方等を書いていくことを目的としています。

## 公式ドキュメントとの違い
公式ドキュメントはアルファベット順で並んでますが、こちらではできるだけ用途別分けました。  
また、サンプルコードをそれぞれにつけています。  

注意点
- あくまでも公式サイトの翻訳ではないです
- deprecationなものは記載しません
- v7.x(Current)に使われているものを中心に書いていきます
